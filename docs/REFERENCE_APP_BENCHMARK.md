# Reference App Benchmark

This tracks CertPath 123 against `../rules/refrence_apps.md`.

## Current Status

| Reference | Target standard | Status | CertPath coverage |
|---|---|---|---|
| Microsoft Learn | Certification workspace, domains, modules, practice, credentials | Partial | Separate certification pages, curriculum, lessons, readiness, resources, credential vault |
| LearnZapp | Diagnostic tests, adaptive study, weak areas | Partial | Weak-domain tracking and readiness exist; diagnostic onboarding remains |
| Pocket Prep | Quick practice, domain practice, missed questions, mocks | Implemented foundation | Practice Centre has Quick 10, topic, domain, weakest-domain, missed-question, custom practice, and certification mode |
| MeasureUp | Separate practice and certification modes | Implemented foundation | Practice mode and timed certification mode are separated in launch flow |
| DataCamp | Learn, practise, assess, apply | Partial | Lessons, quizzes, readiness, and projects exist; ordered skill tracks need deeper UI |
| KodeKloud | Guided labs and evidence | Partial | Project evidence upload exists; lab tasks/checkpoints remain |
| roadmap.sh | Visual roadmap lanes and milestones | Partial | Paid/free lanes exist; dependency/cost/milestone visualisation remains |
| Anki | Due cards and review ratings | Partial | Topic-linked cards with Again/Hard/Good/Easy scheduling exist; retention stats remain |
| Duolingo | Daily goals, streaks, habit loops | Partial | Planner and weekly target exist; streaks/quests/reminders remain |
| W3Schools | Bite-sized lessons, examples, exercises, quizzes | Partial | Lesson reader has key concepts, examples, exercises, quick quiz, references |

## Next Standard Gaps

1. Add diagnostic assessment onboarding for each certification.
2. Add guided lab pages with objectives, tasks, checkpoints, evidence, and reflection.
3. Upgrade roadmap nodes with cost, prerequisites, study time, readiness, and project links.
4. Add habit signals: daily goal, streak, due reviews, and weekly effort summary.
5. Add flashcard retention statistics and suspended card support.
